# Project Scope

## Goals
- Teach car brand recognition to children aged 5–12 through play
- Two complementary game modes: guess-the-brand and guess-the-logo
- Three difficulty levels to grow with the child
- Italian and English language support
- Deploy as a static SPA to GitHub Pages — zero infrastructure required

## Target Users
- Primary: children aged 5–12 (car enthusiast kids)
- Secondary: parents playing alongside their children

## Supported Platforms
- Modern mobile browsers (iOS Safari, Android Chrome) — primary target
- Desktop browsers (Chrome, Firefox, Safari, Edge)
- Minimum viewport: 360px wide

## Out of Scope
- Backend / server-side rendering
- User accounts or authentication
- High-score leaderboards
- Ads or analytics tracking
- Offline PWA (service worker)
- Audio files (sound effects use Web Audio API)
- Timed mode
- Non-automotive brands (motorcycle, truck, etc.)

## Non-goals
- Covering all car brands ever made (limited to 44 well-known brands)
- Real-time multiplayer
- Accessibility beyond WCAG AA
